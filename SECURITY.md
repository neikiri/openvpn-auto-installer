# Security Policy

## 🛡️ Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ Yes     |
| Older   | ❌ No      |

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it responsibly:

* 📧 Email: **[dev@neiki.eu](mailto:dev@neiki.eu)** *(replace this)*
* 💬 Or open a **private GitHub security advisory**

---

## 📋 What to include

Please provide as much detail as possible:

* Description of the vulnerability
* Steps to reproduce
* Affected system (Debian/Ubuntu version)
* Potential impact

---

## ⏱️ Response Time

* Initial response: **within 48 hours**
* Fix timeline: depends on severity

---

## ⚠️ Scope

This project modifies:

* OpenVPN configuration
* iptables firewall rules
* system networking settings

Security issues related to these areas are considered **in-scope**.

---

## 🙏 Responsible Disclosure

We appreciate responsible disclosure and will credit reporters where appropriate.
