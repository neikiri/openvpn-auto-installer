# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [1.0.0] - 2026-04-30

### ✨ Added

* Initial release
* OpenVPN server installation
* OpenVPN server removal
* OpenVPN server reinstall option
* Automatic public interface detection
* Automatic public IPv4 detection
* Client `.ovpn` generation
* NAT configuration using iptables
* IP forwarding setup
* Colored CLI menu
* CLI arguments support (install/remove/reinstall)
* Firewall persistence using iptables-save

---

## 📌 Notes

* This project is intended for Debian/Ubuntu systems
* Use only on servers where you understand networking changes
